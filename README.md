## Дипломный проект. Задание 2: API

### API-тесты для проверки ручек в Stellar Burgers

### Реализованные сценарии

Созданы API-тесты, которые проверяют:
- `Создание пользователя` 
- `Логин пользователя` 
- `Изменение данных пользователя` 
- `Создание заказа`
- `Получение заказов конкретного пользователя`

### Структура проекта
- `tests` - пакет, содержащий тесты, разделенные по классам:
    - `test_creater_user.py` 
    - `test_login_user.py`
    - `test_update_user.py`
    - `test_create_order.py`
    - `test_get_user_order.py`
- `data` - файл, содержащий текстовые ответы
- `conftest` - файл, содержащий фикстуры используемых в тестах
- `requirements` - файл, содержащий список внешних зависимостей
- `allure-results` - пакет, содержащий результаты прохождения отчётов

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`
